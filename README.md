📊 Amazon Data Analysis Project
By Bandi Yaswanth Reddy
📌 Project Overview

The Amazon Data Analysis Project is an end-to-end data analytics and business intelligence project that analyzes Amazon sales data using Excel, MySQL, Power BI, and an AI-based HTML dashboard. The project includes data cleaning, SQL analysis, dashboard development, AI insights generation, and business report creation.

🛠️ Tools & Technologies Used
Microsoft Excel – Data Cleaning
MySQL Workbench – SQL Queries
Power BI – Dashboard
HTML, CSS, JavaScript – AI Dashboard
Chart.js – Visualization
GitHub – Documentation
PDF Report – Insights

📂 Project Workflow
Data Cleaning using Excel
Data Analysis using MySQL SQL Queries
Dashboard Creation using Power BI
AI Dashboard using HTML, CSS, JavaScript
Insights Generation
Business Report Creation.


📊 Key Insights
Total Revenue around $32M
